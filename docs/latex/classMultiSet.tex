\doxysubsection{Detailed Description}
Class representing a multiset of elements. 

The \mbox{\hyperlink{classMultiSet}{Multi\+Set}} class allows for the storage and manipulation of a collection of elements, which can be either strings or nested Multi\+Sets. It supports operations such as addition, removal, and various set operations (union, intersection, difference).
