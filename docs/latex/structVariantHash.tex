Hash functor for std\+::variant containing string or \mbox{\hyperlink{classMultiSet}{Multi\+Set}}. 

This structure provides a way to generate a hash value for std\+::variant objects that can hold either a string or a shared pointer to a \mbox{\hyperlink{classMultiSet}{Multi\+Set}}.
